一、下载 nginx 和 nginx-rtmp源码：
http://nginx.org/download/nginx-1.5.10.tar.gz
https://github.com/arut/nginx-rtmp-module/archive/master.zip

二、编译安装：

  • cd nginx-1.5.10
  • ./configure –with-http_ssl_module –add-module=../nginx-rtmp-module-master
  • make
  • make install
